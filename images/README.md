# Images Folder

This folder contains background images for the website.

## How to add images:

1. Upload your image files here (jpg, png, etc.)
2. The images will automatically be available in your website

## Example images to add:
- `robot.jpg` - for the robotics project
- `swimming.jpg` - for the swim tracker project
- `email-bg.jpg` - for the email button
- `youtube-bg.jpg` - for the YouTube button
- `discord-bg.jpg` - for the Discord button
